📡 WiFi Scanner

An ESP32 project that scans and displays all available WiFi networks

What It Does

The ESP32 scans for WiFi networks in range and displays them on the Serial Monitor with signal strength (RSSI) and security type.

Features

  • ✅ Scans all WiFi networks within range
  • ✅ Shows signal strength (RSSI in dBm)
  • ✅ Displays security type (WPA2, WPA3, Open, etc.)
  • ✅ Shows MAC address of each network
  • ✅ Executes scan every 10 seconds

Hardware

  • ESP32 DevKit
  • USB cable for programming
  • Computer with Arduino IDE or Arduino CLI

Quick Start

  1. Upload wifi_scanner.ino to your ESP32
  2. Open Serial Monitor at 115200 baud
  3. Watch WiFi networks appear in real-time
